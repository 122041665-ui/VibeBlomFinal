# VibeBloom

VibeBloom es una plataforma web diseñada para la exploración y descubrimiento de lugares basada en el ambiente, la experiencia y las preferencias del usuario. El sistema permite consultar, filtrar y recomendar espacios como restaurantes, cafeterías, bares, parques, museos y otros puntos de interés, integrando una interfaz moderna y un asistente inteligente.

---

## Descripción General

El objetivo de VibeBloom es facilitar la toma de decisiones al momento de elegir un lugar, ofreciendo información clara, visual y personalizada. A través de filtros avanzados y el apoyo de inteligencia artificial, el usuario puede descubrir espacios que se ajusten a su presupuesto, ubicación y tipo de experiencia deseada.

---

## Objetivo del Proyecto

Desarrollar una aplicación web moderna que integre:
- Diseño centrado en el usuario
- Búsqueda eficiente y filtrado de información
- Uso de inteligencia artificial para recomendaciones
- Arquitectura clara y escalable
- Buenas prácticas de desarrollo web

---

## Funcionalidades Principales

- Búsqueda de lugares por nombre, ciudad, tipo y rango de precio
- Visualización de lugares mediante tarjetas informativas
- Sistema de calificaciones y reseñas
- Gestión de favoritos
- Carga y visualización de imágenes
- Mapa interactivo de ubicaciones
- Asistente conversacional inteligente para recomendaciones
- Interfaz consistente y responsiva

---

## Arquitectura del Sistema

VibeBloom sigue una arquitectura en capas basada en el patrón MVC:

- **Capa de Presentación**
  - Blade Templates
  - Tailwind CSS
  - Componentes reutilizables

- **Capa de Lógica de Negocio**
  - Controladores Laravel
  - Servicios para IA y ranking de lugares

- **Capa de Datos**
  - Modelos Eloquent
  - Base de datos MySQL

- **Servicios Adicionales**
  - Integración con APIs de inteligencia artificial
  - Envío de correos
  - Almacenamiento de imágenes
